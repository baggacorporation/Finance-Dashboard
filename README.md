# Nextgen Finance Dashboard

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Features
- Net Worth Tracker
- Budgeting Analysis
- Investment Portfolio
- Simulations
- Goals Tracker
- Stock Watchlist (Mock Data)

## Tech Stack
- Next.js
- Tailwind CSS
- Recharts
- Shadcn UI
- Framer Motion
